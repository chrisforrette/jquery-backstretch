Stretch

This is a demo setting the 'constrain' option to 'true'. The image will not be stretched beyond its original dimensions.


<script src="../lib/jquery-1.6.4.min.js"></script>
<script src="../jquery.backstretch.min.js"></script>
<script>
    $.backstretch('pot-holder.jpg', {
        speed: 500,
        constrain: true
    });
</script>
